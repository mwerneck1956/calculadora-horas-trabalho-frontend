> Um site que recebe entrada o numero de hora de chegada e hora de saída de um funcionário e retorna o total de horas diurnas e noturnas trabalhadas do mesmo

## Prerequisite
```sh
node
npm || yarn
```

## Install

```sh
yarn install
```

## Usage

```sh
yarn run start
```
